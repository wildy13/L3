Installation

To begin using the L3 documentation system locally or in your own project, follow the steps below.

1. Download the dist Folder

Visit the GitHub repository and download the contents of the dist folder here:
https://github.com/wildy13/L3/tree/master/docs/dist

Tip: If GitHub doesn't allow downloading a single folder directly, you can:
  • Clone the whole repository using git clone https://github.com/wildy13/L3.git, or
  • Use download-directory.github.io to download only the dist folder.

2. Serve the Folder

To view the documentation locally, serve the dist folder using any static file server. For example: 

npx serve dist

This will open the documentation in your browser (default: http://localhost:3000).

3. Open in Browser

Once served, navigate to the local URL to explore the documentation using the sidebar menu.
