Getting Started

Welcome to the L3 documentation. This guide will help you set up your development environment and prepare to run the included WebVR and WebXR examples.

Required Developer Tools

Important: For local development and testing of WebVR/WebXR features, please install the following Chrome extension:

Web Access Extension – Chrome Web Store

This extension is required to enable proper local file access and WebXR debugging support in Chrome.
